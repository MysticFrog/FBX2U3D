# License Compatibility Notes

This repository uses a mixed-license layout.

The original FBX2U3D project files are released under the repository `LICENSE`, which emphasizes that commercial use requires prior written agreement.

The Intel Universal 3D Sample Software bundled alongside this project remains Apache-2.0 licensed. Apache-2.0 does allow downstream derivative works to use different licensing arrangements as a whole, but the Apache-granted rights for the Apache-covered SDK files themselves still remain in force and must stay identifiable.

In practice, that means:

- the FBX2U3D project's `no commercial use without prior agreement` rule applies to the original FBX2U3D project materials
- the vendored Intel SDK archive and extracted Intel SDK files continue to be governed by Apache-2.0
- downstream redistributors should preserve those license boundaries and notices instead of presenting the Intel SDK as if the noncommercial rule overrides Apache rights

This note is documentation only and is not a substitute for legal advice.
